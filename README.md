# tetoris

"tetoris" is a game created based on the popular "Tetris" game with C, C++.

# Features

tetoris used PDCurses (https://pdcurses.org/).

# Requirement

* ISO C++ 14
* PDCurses v3.9

# Note

I don't test environments under Linux and Mac.
